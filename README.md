## 二次元辣妹 Ani 火成这样,另一个角色已经被遗忘

Grok 的独立 app 最近更新了一个叫「伴侣」的功能。简单说，就是你可以给 AI 机器人选一个虚拟形象，现在有两个选项：一个是二次元风格的辣妹 Ani，另一个是小熊猫 Bad Ruby。
